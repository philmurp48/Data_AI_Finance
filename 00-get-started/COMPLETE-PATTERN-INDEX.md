## ğŸ’ Key Patterns for Consistency

### Across All Pages

1. **Header Section**
   - Logo (Accenture or Greater Than)
   - Application name
   - User profile with avatar
   - AI Search (when applicable)

2. **Navigation**
   - Consistent across all pages
   - Active state clearly shown
   - Proper hierarchy

3. **Page Content**
   - Breadcrumbs for deep pages
   - Page title/heading
   - Main content area
   - Consistent spacing

4. **Colors**
   - Purple (#A100FF) for primary/AI
   - Green for success
   - Red for danger/critical
   - Yellow for warning
   - Blue for info
   - Gray for neutral

5. **Typography**
   - Graphik/Inter for UI
   - Semibold for headings
   - Regular for body
   - Tabular-nums for numbers

6. **Components**
   - Consistent card styling
   - Same button variants
   - Uniform status badges
   - Matching input styles
